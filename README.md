# HR Analytics Dashboard 📊

## Project Overview
Analyzed 1,470 employee records from IBM HR dataset 
to identify key attrition drivers using Excel, SQL 
and Power BI.

## Tools Used
- Microsoft Excel (Data Cleaning & Pivot Tables)
- MySQL (Data Analysis & SQL Queries)
- Power BI (Interactive Dashboard)

## Key Findings
- Overall attrition rate is 16.12%
- Sales department has highest attrition at 20.63%
- Employees who left earned $2,000 less per month
- Employees under 25 have highest attrition at 39.18%
- Very unhappy employees leave 2x more than happy ones
